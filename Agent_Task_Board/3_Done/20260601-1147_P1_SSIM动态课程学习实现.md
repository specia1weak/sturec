#### 注意事项
- 当前 `search_steps_hint=2600`，与 3 epoch 的总步数(~23700)不匹配，调度在 epoch 1 早期即完成全部退火。后续应改为 `search_steps_hint = total_batches × max_epochs` 或在实验中调整
- 原始 SSIM 的双阶段机制（search→retrain）在 retrain 阶段会冻结 curriculum 影响，后续可考虑端到端模式移除此限制
