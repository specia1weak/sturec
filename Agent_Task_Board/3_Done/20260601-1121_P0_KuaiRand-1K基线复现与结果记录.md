#### SSIM 详细日志
- **搜索阶段**: epoch 1 完成搜索，`best_search_auc=0.783969`
- **再训练阶段**: epoch 2-3 冻结掩码模块进行再训练
- **最终 share_rate=1.000**（全部样本共享），说明在 3 epoch 限制下 SSIM 倾向全共享
- **关键行为**: SSIM 的 `share_rate` 在 retrain 阶段为 1.0（全部硬共享），这是因为 `search_epochs=1` 且 `threshold=0.49` 导致的。如果用更多 epoch 或调整阈值，share_rate 会下降

#### 结论：SSIM 是最佳起始 Baseline

基于实际实验结果的排序：
1. **SSIM (0.78581)** — 3 个模型中最高，且在第 3 epoch 仍在上升（未收敛），长训潜力最大
2. **CurriculumSSIM (0.78578)** — 课程调度器首次跑通即达到与 SSIM 几乎持平的水平，sharing_ratio 从 0.8 衰减到 0.2
3. SharedBottom (0.78570) — 强基线，但 domain4 弱
4. STAR (0.78528) — 场景独立参数有效
5. PLE (0.78528) — 低于预期，PLE 优势需更多 epoch 才能体现
6. Pareto (0.78256) — 备选方向 baseline

**推荐**：以 SSIM 作为动态课程学习的起始 baseline，理由：
- 实际 AUC 最高（0.78581），证明其样本选择机制在 KuaiRand-1K 上有效
- SSIM 的二值样本选择（全部共享/不共享）正是动态课程学习要改进的痛点
- 改造点清晰：将二值 `share_mask` 替换为连续 `affinity_weights` + 课程调度
