## 收敛结论

### 🎯 主攻方向：MSR + 动态课程学习 (Dynamic Curriculum Learning)

**核心思路**：SSIM(KDD'25) 证明了"选择高质量共享样本"有效，但它是静态二值选择。我们提出动态课程——训练初期鼓励跨场景共享，后期逐步收敛到场景专有。

**与已有论文的差异点**：
| 对比对象 | 差异 |
|---------|------|
| SSIM (KDD'25) | 静态二值选择（好/不好）；我们是**动态连续衰减 + 课程排序** |
| Curriculum Learning (CV/NLP) | 通用课程学习未考虑场景间关系；我们是**场景感知的课程 + 归属度软分配** |

### 📋 最小实验闭环

| 组件 | 方案 |
|------|------|
| **数据** | KuaiRand-1K（快速验证）+ 全量 KuaiRand（正式实验）|
| **模型** | SSIM (`model=ssim`) 作为 baseline → 扩展为 CurriculumSSIM |
| **训练** | `@examples/kuairand-1k/kuairan1k.py` — 已有完整训练链 |
| **评估** | 已有 `Evaluator` + `DomainFilter`，支持按场景 AUC 评估 |
| **预期结果** | 相比 SSIM baseline，小场景 AUC 提升 ≥ 0.5%，大场景 AUC 不降 |

### 🥈 备选方向：Pareto/Nash 多场景博弈优化

**核心思路**：Improved Accuracy, Declining Orders(KDD'25) 发现多域训练中"精度↑但转化↓"的陷阱。我们将多场景建模为博弈，用 Nash Bargaining 求解均衡。

**最小实验闭环**：
| 组件 | 方案 |
|------|------|
| **数据** | KuaiRand-1K |
| **模型** | `model=pareto`（已有集成）+ Nash Bargaining Loss 扩展 |
| **训练** | 同 KuaiRand 训练链 |
| **预期结果** | 相比普通 Pareto，所有场景的 Trade-off 更均衡 |

## 第一阶段实验计划 (1-2 周)

### 第 1 天：SSIM Baseline 复现
- [ ] 在 KuaiRand-1K 上运行 SSIM baseline (`model=ssim`)，记录各场景 AUC
- [ ] 再跑 SharedBottom、PLE、STAR 作为对比基线
- [ ] 确认 SSIM 复现结果与论文趋势一致

### 第 2-3 天：场景归属度软分配模块
- [ ] 实现`SceneAffinityLoss`：为每个样本学习软性的"场景归属权重向量"
- [ ] 在 SSIM 样本选择的基础上，加权而非二值地控制跨场景共享程度
- [ ] 在 KuaiRand-1K 上验证归属度权重的分布是否合理

### 第 4-5 天：动态课程调度
- [ ] 实现课程调度器：训练步数 → 衰减函数控制共享程度
- [ ] 与 SceneAffinityLoss 组合：初期低阈值广共享，后期高阈值收窄
- [ ] 在 KuaiRand-1K 上做消融实验

### 第 6-7 天：全量实验 + 分析
- [ ] 切换到全量 KuaiRand 做正式实验
- [ ] 按场景粒度分析 AUC 变化
- [ ] 输出场景归属度权重的可视化分析

### 第 2 周：强化实验 + 备选方向启动
- [ ] 在 Amazon 多域数据集上验证泛化性
- [ ] 与 MMLRec Benchmark 对比
- [ ] 备选方向（Pareto/Nash）启动 baseline 跑通

### 风险点与应对

| 风险 | 可能性 | 应对 |
|------|--------|------|
| 归属度权重退化为均匀分布 | 中 | 加入熵正则化，强制稀疏 |
| 课程调度在 KuaiRand 上收益有限 | 中 | 切换到 Amazon 多域（场景差异更大）|
| SSIM baseline 复现结果与论文不一致 | 低 | 参考 `reference-projects/SSIM-main/` 官方代码调参 |
| 全量 KuaiRand 数据加载慢 | 低 | 使用 ParquetStreamDataset 已支持流式加载 |
